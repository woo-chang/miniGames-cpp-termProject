# miniGames-cpp-termProject
## 프로젝트 설명
* 콘솔창 기반의 미니 게임 모음
* C++ 프로그래밍 학습에 있어 단순한 언어 학습에 그치지 않고 개발을 경험해보고자 진행한 팀 프로젝트
* Visual studio 사용<br/><br/>
## 게임 설명
* 게임을 시작하면 시작 화면으로 이동
* 사용자 계정을 등록 후 게임 점수 기록 및 유지
* 4가지 게임 존재
  * 핑퐁 게임
  * 카드 찾기 게임
  * 뱀 게임
  * 점프 게임
